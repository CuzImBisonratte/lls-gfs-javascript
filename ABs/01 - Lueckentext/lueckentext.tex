Javascript ist \blank{dynamisch} typisiert. Das bedeutet, dass Variablen nicht explizit als Zahlen, Strings oder Booleans deklariert werden müssen. Der Typ einer Variablen wird automatisch durch den \blank{Wert / Inhalt} ermittelt.

Die typisierung von Variablen in Javascript ist nicht \blank{statisch}. Das bedeutet, dass der Typ einer Variablen während das Programm läuft verändert werden kann.

\spacer

\begin{table}[ht]
	\centering
	\begin{tabular}{|c|c|}
		\hline
		\textbf{Aussage}                                         & \textbf{Korrekt?} \\
		\hline
		Es gibt For-Schleifen in JavaScript                      & $\Box$            \\
		If funktioniert gleich wie in C++                        & $\Box$            \\
		Javascript benötigt ein Hauptskript (int main)           & $\Box$            \\
		Javascript muss eingerückt werden, damit es funktioniert & $\Box$            \\
		\hline
	\end{tabular}
\end{table}

\spacer

Zeilen enden wie auch in C++ mit einem \blank{Semikolon}. Variablen werden mit \blank{var} und konstanten mit \blank{const} deklariert. Funktionen werden mit \blank{function} deklariert.

\vspace{1cm}

Javascript kann man in \blank{Dateien} auslagern, welche man dann in der HTML-Datei einbindet. Man kann JavaScript aber auch direkt in der HTML-Datei schreiben, indem man ein \blank{<script>}-Element an das Ende des \blank{<body>}-Elements einfügt.
